Detailed Description

Encapsulates the private implementation interface used by the base class argpp::ArgppBase
(a design pattern known as the Pimpl idiom)

If C++11 or C++0x language support is available, our code uses a std::unique_pointer<T> in the argppbaseimpl.cpp file to hold the pointer to the singleton root parser instance. Otherwise, we use a very simple auto pointer for that which is default constructible with a NULL pointer and never uses copying or assignment.
